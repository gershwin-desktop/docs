# Sharing

The **Sharing** preference pane allows you to manage network sharing services.

:::{note}
As of January 25, 2026 this only partially functional on some operating systems.
:::

## Features
* **Local Discovery**: Enable or disable Zeroconf (mDNS) discovery.
* **File Sharing**: Configure SFTP or other file sharing services.
* **Remote Desktop**: Enable and configure VNC or other remote access protocols.
