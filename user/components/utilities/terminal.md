# Terminal

**Terminal** is Gershwin's terminal emulator. It provides a command-line interface for interacting with the underlying FreeBSD system.

## Features
* **GNUstep Integration**: Built using the GNUstep framework for a consistent look and feel.
* **Customizable**: Supports various terminal settings and color schemes.
* **Services Support**: Supports GNUstep Services to interact with other applications.
