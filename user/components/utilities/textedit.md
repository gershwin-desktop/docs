# TextEdit

**TextEdit** is a simple text editor for Gershwin, based on the GNUstep TextEdit application.

## Features
* **Rich Text Support**: Can edit both plain text and rich text files.
* **Undo/Redo**: Full support for undoing and redoing actions.
* **Spell Checking**: Integrated spell checking support.
* **Font and Typography**: Advanced font management and typography features.
